# Clip Experience Core Foundations

Every clip in the cAMP Clips Path follows these 7 non-negotiable foundations:

1. **Watch Clip** — Wistia-embedded video on the `/watch/:clipId` page
2. **Pause & Resume** — Learners can pause mid-clip and return later. Resume modal offers "Resume" or "Restart" options.
3. **Info Header** — Summary, learning objectives, and SMEs displayed at the top (can be dismissed). Transcript button available.
4. **Engagement Scoring** — Focus (30%) + Time (45%) + Trail Markers/Questions (25%) = Overall Engagement Score
   - **80%+ overall** = clip passes
   - **Below threshold** = S&R (Search & Rescue) triggers with rescue questions
   - **End of clip** = WtS (Weather the Storm) comprehension check with 2-minute timer
5. **Review Ranger Report** — Displayed once clip is complete, showing all performance data
6. **Sequential Unlock** — Completing one clip unlocks the next. For multi-clip days (e.g. Day 1 Industries → Personas), each clip gates the next.
7. **Full Analytics Tracking** — Sessions, engagement scores, XP, badges, pacing, trail marker responses, S&R/WtS results — all tracked and visible in admin analytics
